slid
==========

Vue 3 + TypeScript + Pinia + Tailwind + Vite frontend for **Slid**.

This is a workspace of the [Slid monorepo](../README.md). All commands
are run from the repo root:

````
$ npm install        # at repo root
$ npm run dev        # starts this app
$ npm run test:app   # Jest with coverage
````

The design philosophy and "how to add a layout/theme without regressing quality"
guide lives in [DESIGN.md](./DESIGN.md).
